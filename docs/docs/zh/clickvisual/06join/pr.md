# 如何参与代码贡献

## Fork 阶段
1. fork 代码
2. git clone
3. git remote add upstream https://github.com/clickvisual/clickvisual

## 开发阶段
1. git fetch upstream
2. git rebase -i upstream/master
3. git add .
4. git commit
5. git push

## PR 阶段
点击 "Compare" 比较变更并点击 "Pull request" 提交 PR
